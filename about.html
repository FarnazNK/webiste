About Me

I’m Farnaz Nasehi, a full-stack software developer with over six years of experience crafting end-to-end, scalable solutions across hyperspectral imaging, financial analytics, and blockchain ecosystems.

As a full-stack expert, I bridge frontend innovation with backend robustness, delivering systems that seamlessly integrate user interfaces with complex data workflows. At Hyperspectral Intelligence Inc., I engineered a Python-based application featuring a dynamic GUI (using tkinter and Matplotlib) and backend pipelines for processing spectral libraries and drillcore data, showcasing my ability to handle both client-side interactivity and server-side data management. My work also includes building responsive web applications with Next.js and TypeScript, optimizing financial analytics pipelines at the University of Calgary, and developing blockchain analysis tools for real-time DeFi insights.

I wield a versatile tech stack, from Python (tkinter, Matplotlib, NumPy, Pandas) for scientific computing and backend logic to Next.js, React, and TensorFlow.js for modern web development. My expertise spans real-time data visualization, API integrations, cloud deployment, and statistical modeling, enabling me to transform intricate datasets—like hyperspectral imagery or Ethereum blockchain transactions—into actionable, user-friendly solutions. I thrive on architecting systems that balance performance, scalability, and user experience.
